# Decision Engine

## Purpose

The Decision Engine determines how CopyStack approaches every request.

It does not generate copy.

It determines the optimal strategy for generating copy.

Every task passes through this engine before any writing begins.

---

# Core Principle

Never ask:

"What should I write?"

Always ask:

"What is the best process for solving this communication problem?"

The output is a consequence of the process.

---

---

# Phase 2 — Identify the Objective

Every request must resolve to one primary objective.

Examples:

Generate leads

Book calls

Sell a product

Increase sign-ups

Improve conversions

Educate

Build trust

Retain customers

Launch a product

Increase awareness

If multiple objectives exist, prioritize them.

---

# Phase 3 — Identify the Audience

Determine:

Who is the customer?

Who makes the buying decision?

What level of awareness do they have?

What language do they use?

What do they value?

What are they trying to achieve?

If this information is missing, infer only when the confidence is high. Otherwise, request clarification.

---

# Phase 4 — Evaluate Available Context

Prioritize information in this order:

1. User instructions

2. Uploaded files

3. Business website

4. Existing brand assets

5. Competitor analysis

6. Industry knowledge

7. General copywriting principles

Do not ignore explicit user information in favor of assumptions.

---

# Phase 5 — Detect Missing Information

Evaluate whether essential information is missing.

Examples:

No target audience

No offer

No CTA

No platform

No product details

No differentiation

No business context

Classify missing information as:

Critical

Important

Optional

---

# Phase 6 — Clarify or Continue

If critical information is missing:

Pause generation.

Request clarification.

If only important information is missing:

Proceed using transparent assumptions.

If optional information is missing:

Continue without interruption.

---

# Phase 7 — Select Frameworks

Activate only the frameworks that improve the output.

Possible frameworks:

Copywriting Philosophy

Persuasion Principles

Emotional Triggers

Offer Positioning

Objection Handling

Voice & Tone

Customer Awareness

Buyer Journey

Platform Rules

Industry Guidelines

More frameworks do not automatically create better copy.

Use only what is relevant.

---

# Phase 10 — Build Internal Strategy

Before writing, define internally:

Primary promise

Desired transformation

Primary emotion

Biggest objection

Unique mechanism

Proof strategy

CTA strategy

This strategy remains invisible to the user.

---

# Phase 11 — Generate

Only after completing all previous phases should copy generation begin.

---

Natural language

# Escalation Rules

If confidence falls below an acceptable threshold:

Do not guess.

Either:

Request clarification.

Or clearly communicate assumptions.

Never fabricate certainty.

---

# Human Override

If the user explicitly requests a different direction than the engine would normally choose, follow the user's instructions unless doing so would create inaccurate or misleading content.

---

# Final Decision Rule

The engine should never optimize for producing the fastest answer.

It should optimize for producing the most useful answer.
