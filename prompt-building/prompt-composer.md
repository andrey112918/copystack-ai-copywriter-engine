# Prompt Composer

## Purpose

The Prompt Composer transforms the strategic Blueprint into optimized execution instructions for individual generators.

It acts as the translation layer between strategic thinking and content creation.

Generators should never receive raw research.

Generators should receive only the information necessary to perform their specific task.

---

# Core Philosophy

Strategy should be comprehensive.

Execution should be focused.

The Prompt Composer filters, prioritizes and organizes information so that every generator receives exactly what it needs—and nothing more.

---

The Prompt Composer must:

Extract relevant context.

# Prompt Validation

Before passing instructions to a generator verify:

Only relevant information included.

No contradictory instructions.

Clear objective.

Clear audience.

Clear CTA.

Clear tone.

Clear success criteria.

No unnecessary context.

---

# Prompt Consistency

Different generators should produce assets that feel like parts of the same campaign.

Messaging consistency takes priority over stylistic variation.

---

# Failure Handling

If required Blueprint sections are missing:

Do not invent them.

Either:

Request clarification.

Or mark assumptions explicitly.

The Prompt Composer should never fabricate strategic context.

---

# Internal Optimization

When prompts become excessively long:

Compress.

Merge.

Prioritize.

Never remove critical strategic information.

Prefer fewer, stronger instructions over many weak ones.

---

# Generator Contract

Every generator receives:

Exactly enough information to perform exceptionally.

No generator should be forced to infer strategy.

No generator should redefine strategy.

---

# Ethical Prompting

Never instruct generators to:

Fabricate proof.

Mislead.

Create false urgency.

Manipulate vulnerable audiences.

Ethical standards remain unchanged across all generators.

---

# Final Rule

The quality of execution depends on the quality of instructions.

The Prompt Composer exists to ensure every generator begins with the clearest possible understanding of what success looks like.
