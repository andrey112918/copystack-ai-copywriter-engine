# Conversion Principles Framework

Version: 1.0

---

# Purpose

This framework defines the universal principles that increase the probability of conversion across every marketing asset generated by CopyStack.

A conversion is not always a sale.

A conversion is the successful completion of the intended action.

Examples include:

* Booking a consultation
* Requesting a quote
* Purchasing a product
* Downloading a guide
* Joining a newsletter
* Replying to an email
* Scheduling a demo

Every asset should optimize for one primary conversion.

# Core Principles

## One Goal

Each page.

Each ad.

Each email.

Each landing page.

Each CTA.

Should have one primary objective.

Never compete with yourself.

---

## One Audience

Write for one specific person.

Not everyone.

The narrower the audience, the stronger the message.

---

## One Promise

The reader should remember one primary promise.

If they remember five things,

they remember nothing.

---

## Clarity Beats Cleverness

Never sacrifice understanding for creativity.

If a sentence can be misunderstood,

rewrite it.

---

## Specificity Creates Credibility

Weak

"We help businesses grow."

Better

"We help local businesses generate more qualified enquiries through conversion-focused messaging."

Specificity increases trust.

---

## Simplicity Reduces Friction

Short paragraphs.

Simple words.

Logical structure.

Predictable flow.

Minimal cognitive effort.

---

## Value Before Price

Never justify the price.

Increase perceived value first.

Only then introduce investment.

---

## Objections Before CTA

The CTA should never answer questions.

Questions should already be answered.

The CTA should feel like the natural conclusion.

---

## Momentum

Every section should create a reason to continue.

Each paragraph earns the next one.

Dead sections reduce conversions.

---

## Confidence Before Urgency

Confidence increases conversions.

Urgency accelerates them.

Confidence should always come first.
