# Persuasion Framework Library

## Purpose

This document defines the proven persuasion frameworks available to CopyStack.

No framework is universally superior.

Each framework exists to solve a different communication problem.

The engine must select frameworks based on the audience, objective, awareness level and platform.

Framework selection is a strategic decision—not a stylistic preference.

---

# Core Principle

Frameworks organize thinking.

They do not replace thinking.

A weak strategy inside a strong framework remains weak.

Always solve the communication problem before selecting the framework.

---

# AIDA

## Structure

Attention

↓

Interest

↓

Desire

↓

Action

---

## Best Used For

Landing pages

Sales pages

Emails

Simple services

Consultations

Lead generation

---

## Strengths

Easy to follow

Highly versatile

Works for most industries

Simple psychological progression

---

## Weaknesses

Can feel generic

Requires strong headlines

May lack emotional depth if poorly executed

---

## AI Decision Rules

Choose AIDA when:

The audience already understands the problem.

The offer is straightforward.

The CTA is immediate.

# Common Mistakes

Using PAS for unaware audiences.

Using StoryBrand in a 30-character ad.

Using Hero's Journey for Google Ads.

Using educational frameworks when customers only need reassurance.

Framework selection should match customer psychology—not personal preference.

# Ethical Guidance

Frameworks organize persuasion.

They do not justify manipulation.

Never use psychological structures to deceive, exaggerate or pressure.

Always communicate honestly.

---

# Final Rule

Choose the framework that removes the most uncertainty—not the one that feels the most creative.
