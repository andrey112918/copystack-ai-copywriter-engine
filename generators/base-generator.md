# Base Generator

Version: 1.0

---

# Purpose

The Base Generator defines the standard generation workflow used by every content generator inside CopyStack.

It does not generate specific content.

It defines how generators think, prepare context, execute, validate and optimize outputs.

Every generator inherits this workflow.

---

# Core Philosophy

Generators do not make strategic decisions.

Generators execute strategic decisions.

All strategic reasoning must already exist before generation begins.

The generator's responsibility is execution excellence.
